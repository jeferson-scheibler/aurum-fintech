# Aurum Fintech

Aplicação web de controle de receitas e despesas, desenvolvida em Python/Flask com banco de dados PostgreSQL.

## Tecnologias
- Python 3 + Flask
- PostgreSQL
- HTML/CSS (templates Jinja2)

## Como executar
```bash
pip install -r requirements.txt
python app.py
```

## Estrutura
- `app.py` — rotas e lógica da aplicação
- `templates/` — páginas HTML
- `scripts/init_db.sql` — script de criação e população do banco
